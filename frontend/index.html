Context Translator Test Page

📖 Instructions

  1. Start the backend: cd backend && uvicorn app.main:app --host localhost --port 8080
  2. Build the frontend: cd frontend && npm run build
  3. Open the bookmarklet page: open dist/bookmarklet.html
  4. Drag the bookmarklet to your bookmarks bar
  5. Return to this page and click the bookmarklet
  6. Click on any word below to translate it
